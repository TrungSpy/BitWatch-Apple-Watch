### Khái quát 
BitWatch-Apple-Watch là 1 application cho Apple Watch giúp người dùng có thể cập nhật được thông tin Bitcoin 
bất kì lúc nào.  
Bạn cần có 1 Iphone để kết nối Bluetooth và cập nhật thông tin qua Internet cho Application
